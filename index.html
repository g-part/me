The worlds of our craft

My favorite editor: The Hammer Editor. Most of my work in the past was done in Source 1 games using the Hammer Editor by Valve. I personally loved how easy it was to create maps. The blocking tools of the editor allowed me to create different kinds of structures without using additional tools like a 3D modelling software.
